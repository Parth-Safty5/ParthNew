# ✍️ ParthNew – Writing Skills Hub

**ParthNew** is an educational website that helps students improve their **English writing skills**.  
The platform provides structured guides, writing rules, and ready-to-use templates for different types of academic writing.

The goal of this project is to make writing simple and accessible for students by offering **clear formats, examples, and practical templates**.

---

## 🌐 Live Website

Visit the live website hosted on Vercel:

🔗 https://parth-new.vercel.app

---

## 📚 Writing Topics Available

ParthNew includes many important writing skill topics such as:

- 📝 Essay Writing
- ✉️ Letter Writing
- 📊 Report Writing
- 📄 Application Writing
- 📩 Invitation Letters
- 📚 Writing Formats and Rules
- 📋 Writing Templates for Students

## Each topic contains:

- Clear format explanation  
- Writing rules  
- Practical examples  
- Ready-to-use templates
